YggMesh

View on GitHub
A self-organizing mesh network for when the grid goes down.

YggMesh is open-source firmware that turns low-cost travel routers into a self-organizing, zero-configuration encrypted network.

Nodes discover each other automatically and start routing the moment they power on. The moment at least one node gets internet access, the whole network does too.

Scenarios

  • Cellular Blackouts & Grid Failures: Rapidly deployable local IP communications for emergency response or when primary telecom infrastructure goes completely dark.
  • Overlanding & Off-Grid Camping: Link vehicles, trucks, and basecamps together to seamlessly share local maps, secure chats, and files where cellular networks don't exist.
  • Cooperative Local Broadband: Drop nodes across a site to automatically build a resilient network, securely pooling and sharing a single internet uplink (like a satellite or mobile hotspot) across all users.
